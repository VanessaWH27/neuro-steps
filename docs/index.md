# Fundamentals of AD and Neuroscience Research

This portal is a student-friendly knowledge base for learning Alzheimer's Disease (AD) research, with clear math explanations and simple examples.

## How to use this site
- Start with **AD Basics**
- Then review **Math & Methods**
- Follow the **MRI & Machine Learning** workflow pages
- Use **Resources** for datasets and a glossary
